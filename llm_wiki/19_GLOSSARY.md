# Glossary

- `substrate`
  a lower-level evidence-producing layer

- `meta layer`
  a layer that interprets or aggregates prior evidence

- `silent fallback`
  degrading to a weaker method without exposing it in outputs or metadata

- `research-faithful`
  implementation materially matches the cited method, not just the name

- `practical approximation`
  useful engineered proxy that does not fully implement the research method

- `heuristic baseline`
  lexical, regex, threshold, or rule-cascade logic

- `evidence producer`
  component that creates structured evidence records or findings

- `evidence interpreter`
  component that maps, aggregates, ranks, or explains evidence

- `danger zone`
  high-coupling file where small edits can distort system behavior broadly

- `context engineering`
  the discipline of preserving accurate architectural context across code, docs, tests, reports, and agent workflows so that future work stays grounded and non-destructive

- `documentation drift`
  the state where the wiki or docs no longer match the implementation and start misleading engineers or agents
