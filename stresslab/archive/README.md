# Stresslab Archive

This directory holds legacy stresslab assets that are intentionally kept out of the active
runtime paths to reduce confusion.

Active claim-diagnosis gold sets:
- `stresslab/cases/golden/claim_diagnosis_gold_v0.json`
- `stresslab/cases/golden/claim_diagnosis_gold_v1.json`

Archived here:

- calibration artifacts that are not part of the active claim-diagnosis flow
- baseline validation reports kept for historical reference only
